## CDN

> 写简单页面时,尽量用以下 CDN 资源, 如 axios, vue 的引用, 正式上线的时候，把资源下载到本地，或者传到CDN 

- [jsdelivr](https://www.jsdelivr.com/)  // npm 关联 cdn, 文件全，个人开发可用，如对外展示，可换成bootcdn
- [bootcdn](https://www.bootcdn.cn/)  // 境内CDN，速度快
